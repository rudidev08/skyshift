## Principles

### Accessible and runs everywhere

Browser, light-weight library if needed. Don't prematurely invest in game engine until needed, since it adds barriers to try the game out (open web link vs install an app).

### Living world

World that changes and evolves outside player interaction. Player can participate, or just observe. Player participation is not yet implemented.

### Joyful

It's not a screensaver, but should be pleasing to look like it's one!

## Phaser

Built on [Phaser](https://phaser.io) v4.0.0. Run `dev/fetch-phaser-docs.sh` to generate local API documentation in `dev/phaser-docs.local/`.
